# Guitar Hero III Leaderboards
## Summary
A leaderboard app made in Blazor to track high scores for songs on the Guitar Hero III setlist.
## Installation/How to Run
1. Extract file contents to a folder for safe keeping.
2. Open the solution using Visual Studio.
3. Run with or without debugging.
## Conclusion
In all, we should have definitely used a different option for our DB. Because it is made in SQLite, we cannot deploy.
Either way, feel free to fork this is you want to make a truly deployed DB!
